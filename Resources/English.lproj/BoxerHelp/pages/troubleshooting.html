Solving common problems

  • My game runs too fast or too slow

  • My game has multiple CDs, how do I swap between them?

Keyboard and mouse issues

  • I have a one-button mouse, but my game needs two buttons
  • The function keys don’t work
  • I keep switching OS X Spaces while trying to play

If you didn’t find an answer here, contact support with the details of the problem and I’ll help you out as best I can. Be sure to mention which game and where you got it from.
